## Bootstraping Your App

Setup your app by adding GPT API key in `config/local_env.yml`:
```yml
OPENAI_API_KEY: sk-XXXXX
```

Then install the ruby dependencies and spin up the server:
```sh
$ bundle install
$ bin\rails server
``
